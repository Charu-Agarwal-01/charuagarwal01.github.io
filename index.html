This project tackles semantic segmentation of high-resolution satellite imagery using UNet and VGG19-UNet to accurately classify six land covers including vegetation, water, buildings, lands and roads. Demonstrated a full end-to-end pipeline from data preprocessing to model evaluation.

Energy Demand Forecasting - Time Series Forecasting

This project creates a system to predict hourly electricity usage on a monthly basis using historical data. I explored and compared three different models: ARIMA, Random Forest, and LSTM, and fine-tuned them to improve accuracy. The best-performing model was then used to create a Streamlit web application, making the predictions easy for anyone to access and use.
